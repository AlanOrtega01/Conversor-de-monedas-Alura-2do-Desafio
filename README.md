# CONVERSOR DE UNIDADES
*Programa creado por Adonnai Sosa*

Este programa convierte unidades*.
las unidades aceptadas son:

- Divisas: dolar americano, peso mexicano, yuan japones, won surcoreano y libra esterlina.
- Masas: kilogramos, libras, gramos, litros.

**NOTA*
Por el momento solo funciona con divisas, proximamente estaran disponibles las nuevas funcionalidades.*
